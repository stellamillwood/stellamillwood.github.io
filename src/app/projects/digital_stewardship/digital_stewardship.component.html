Context & Goal

The research approached waste management as a collective and infrastructural practice and explored how real-time data and coordination tools could support residents in recycling more effectively. The interface was developed from interviews and workshops.

Final refined UI on mobile.

My Contributions

  • Designed the interface and user journey in Figma
  • Developed the web front end with JavaScript, HTML, and CSS
  • Used Git for version control and backup
  • Mounted sensors and supported data integration
